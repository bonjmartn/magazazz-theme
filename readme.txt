------------------------------
MAGAZASS THEME
------------------------------

Thanks for using the Magazazz theme. This is a minimal and modern magazine-style blog theme with a grid layout and support for multiple authors.

The theme's main informational page is at:
https://www.zenwebthemes.com/shop-themes/magazazz/

------------------------------
FREE AND PREMIUM VERSIONS
------------------------------

This theme comes in two versions: Free and Premium.

Free Features

- Responsive design
- Theme customizer
- Custom logo
- Social media icons in footer
- Grid layout for blog posts on homepage and archives
- Large featured images on blog posts
- Documentation

Premium Features

The Premium version has all of the free features, plus:

- Support for multiple authors
- Choose your own accent color for links and hover effects
- Choose from 14 Google Fonts for headings and paragraph texts
- Choose paragraph text size
- Choose colors for text and headings
- Homepage headline
- Two open widget content areas on the homepage
- End of Posts widget
- Choose from Left Sidebar, Right Sidebar, and Full Width page layouts
- Professional tech support
